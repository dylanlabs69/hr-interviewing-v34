## What hr-interviewing does

hr-interviewing is a browser-first HR interview guide generator that converts a resume and a job description into an organized set of interview questions. It is built to emphasize candidate-role alignment, making it easier to focus discussions on the experience, skills, and expectations that matter most.

Since the app runs entirely in the browser, it fits teams that want a compact, single-page process without installs, build steps, or any server deployment. Recruiters, interviewers, and hiring coordinators can use it to review qualifications, spot gaps, and prepare sharper interviews in less time.

---

## Core capabilities

- Builds structured interview guides from resumes and job descriptions
- Creates questions tailored to the candidate and the role
- Highlights matching qualifications and missing requirements
- Lets you print or save interview guides as PDF
- Operates as a single-page browser app
- Processes data locally with no server setup
- Keeps HR prep straightforward and fast
- Works well for repeatable interview planning across multiple roles

## How to use it

1. Open the app in your browser.
2. Paste or load the resume/CV content.
3. Add the job description for the role.
4. Generate the interview guide.
5. Review the suggested questions, fit highlights, and qualification gaps.
6. Print or save the result as a PDF for interview-day use.

Typical workflow:

- Prepare one candidate profile
- Compare it against one role description
- Generate targeted questions
- Export the guide for the hiring team

## FAQ

**How do I get updates?**  
Use the latest published build link, or pull the newest repository changes if you are working from a local clone.

**Where are settings stored?**  
In this browser-based setup, configuration lives client-side inside the app or the browser session.

**What if the guide does not generate as expected?**  
Refresh the page, verify that the resume and job description were loaded correctly, and try again with cleaner text input.

**Can I use it offline?**  
After the files are stored locally, it is designed to run as a single-page browser app with no server setup.

**How do I save the output?**  
After generating the interview guide, use the built-in print or PDF export option.
